
# Simple Music Player
A simple web-based music player for local audio files made by m1rabl1z.



## Features:

- Upload/Select audio files locally
- Display the name of current song
- Fullscreen mode
- Basic controls:
    * Play
    * Pause
    * Next
    * Previous
- A working progress bar that updates as the songs plays
- Loop toggle(Single/Multiple)
- Proper handling when no songs selected
- Rudimentary Shuffle

## Takeaways/Insights:
- Interplay of HTML, CSS and JavaScript
- Basic file-handling
- Basic webpage-styling
## Run Locally

 - Clone the project

```bash
  git clone https://github.com/m1rabil1z/b29-html-css-js-assgn.git
```

- Go to the project directory

```bash
  cd b29-html-css-js-assgn
```

- Open in VSCode, and run the index.html file using the Live Server Extension
